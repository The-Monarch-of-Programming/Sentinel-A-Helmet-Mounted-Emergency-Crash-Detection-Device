Project Title: Sentinel-A-Helmet-Mounted-Emergency-Crash-Detection-Device

Description: 
The Helmet-Mounted Emergency Alert Mobile Application is a safety-focused app designed to work alongside a smart helmet IoT device. The application acts as the central platform for receiving accident alerts, displaying real-time location data, and notifying emergency contacts when an incident occurs.

Technologies Used: Flutter, Dart, Arduino, GitHub, Cloud Backend

Features:
- Receives accident and motion data from the helmet-mounted IoT device
- Processes sensor data to determine alert severity
- Automatically sends push notifications or SMS alerts to emergency contacts
- Displays incident location on an interactive map
- Records timestamps and sensor readings for each alert
- Allows users to manually send or cancel alerts
- Syncs device data with a cloud backend for reliability and accessibility
